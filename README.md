<h2 align="center">Re:京都産業大学マップ</h2>
現在の学内マップは使いづらい為、「構内の建物へ案内する為の足跡を作成し実際に地面に貼り付けて道案内をする」というプロジェクトで作成したシュミレーションソフトウェアです。
<br><br>
足跡の長さは27.0cm、足跡の幅は9.0cmです。

## 📦 Features

<h3 align="center">- 地面生成 -</h3>

<p align="center">
  <img src="https://user-images.githubusercontent.com/60394438/103476603-b0e6a380-4dfa-11eb-9404-beaf6bac5180.gif" width=100%>
</p>
座標(0,0,0)を原点とし、右と奥にタイルをそれぞれ何枚敷き詰めるかをパラメータで指定できます。
<br>

<h3 align="center">- 地面の種類を変更できます -</h3>

<p align="center">
  <img src="https://user-images.githubusercontent.com/60394438/103475968-2780a280-4df5-11eb-9808-f435d25f3d37.png" width=40%>
  <img src="https://user-images.githubusercontent.com/60394438/103475973-2ea7b080-4df5-11eb-9896-586bfc08ecfe.png" width=40%>
  <img src="https://user-images.githubusercontent.com/60394438/103475976-323b3780-4df5-11eb-86b5-929dbfb1eba7.png" width=40%>
  <img src="https://user-images.githubusercontent.com/60394438/103475977-35362800-4df5-11eb-9cf5-5327692c26cf.png" width=40%>
</p>
地面のテクスチャを変えることで、様々な場所で視認性のシュミレーションができます。
<br>

<h3 align="center">- シュミレーションをして制作費のコスト削減 -</h3>

足跡はPNG形式なので、チームのデザイナーが用意してくれた足跡を素早く試せます。またシュミレーションを行うことで実際に作った時のミスマッチを減らせます。
<br>

<h3 align="center">- 風景や明るさも変更できます -</h3>
明るい場所にある黄色の足跡の見え方
<p align="center">
  <img src="https://user-images.githubusercontent.com/60394438/103476170-c3f77480-4df6-11eb-9475-e913c970e84f.png" width=100%>
</p>
暗い場所にある黄色の足跡の見え方
<p align="center">
  <img src="https://user-images.githubusercontent.com/60394438/103476172-c9ed5580-4df6-11eb-853f-ce0e2dd87c96.png" width=100%>
</p>
UnityのSkyboxやライトを変更することで、空間の明暗を変える事ができます。また取り込んだ足跡は拡大縮小など変形も可能です。
<br>
